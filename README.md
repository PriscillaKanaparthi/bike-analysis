
# 🚴‍♂️ Bike Sharing Demand Analysis (EDA Project)

## 📌 Project Overview
This project explores a Bike Sharing dataset using Exploratory Data Analysis (EDA) techniques to understand user behavior, demand patterns, and key factors influencing bike usage.

The objective is to extract actionable insights that can help improve operational efficiency, resource allocation, and user experience in a bike-sharing system.

---

## 🎯 Objectives
- Analyze bike usage patterns across time and user types  
- Identify peak demand periods  
- Understand user segmentation (subscribers vs casual users)  
- Perform geospatial analysis of ride locations  
- Generate business insights for optimization  

---

## 🛠️ Tools & Technologies
- Python 🐍  
- Pandas & NumPy  
- Matplotlib & Seaborn  
- Folium (Geospatial visualization)  
- Jupyter Notebook  

---

## 📊 Key Analyses Performed

### 1. Trip Duration Analysis
- Distribution of ride durations  
- Identification of short-trip dominance  

### 2. User Behavior Analysis
- Gender-based distribution  
- Subscriber vs casual user comparison  

### 3. Time-Based Analysis
- Hourly usage patterns  
- Weekday vs weekend trends  
- Peak demand identification  

### 4. Feature Engineering
- Extracted hour, weekday/weekend features  
- Created time-based grouping variables  

### 5. Geospatial Analysis (Folium)
- Visualized ride locations on interactive maps  
- Identified high-demand geographic zones  
- Analyzed spatial clustering of usage  

---

## 💡 Key Insights
- Most rides are short-duration trips (commuting behavior)  
- ~92% of rides are from annual subscribers  
- Clear peak hours during weekdays (rush-hour pattern)  
- Weekend usage is more evenly distributed  
- Demand is highly time-dependent and predictable  

---

## 📌 Business Recommendations
- Increase bike availability during weekday peak hours  
- Optimize station placement in high-demand zones  
- Target marketing campaigns toward casual users  
- Improve inventory planning using time-based demand trends  

---

## 🗺️ Geospatial Insights
Folium maps were used to visualize ride distributions, revealing:
- High-density usage clusters  
- Key mobility hotspots  
- Spatial demand imbalance across regions  

---

## 🚀 Outcome
This project demonstrates how raw mobility data can be transformed into meaningful business insights using EDA and visualization techniques.
