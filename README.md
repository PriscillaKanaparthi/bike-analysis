# bike-analysis
“This project analyzes Citi Bike usage patterns to identify high-demand stations and temporal trends in bike usage.”
🚲 Bike Usage Pattern Analysis

This project explores bike-sharing usage patterns by analysing hourly ride data across weekdays and weekends.

🎯 Objective

To identify behavioural differences in bike usage based on time of day and day type.

🛠 Tools & Technologies
Python
Pandas
Matplotlib
Jupyter Notebook
📊 Key Insights
Weekday usage peaks during morning and evening commute hours
Weekend usage is more evenly distributed throughout the day
Clear behavioural differences suggest commuting vs leisure usage patterns

📁 Skills Demonstrated
Data cleaning and preprocessing
Feature engineering (time-based features)
Exploratory data analysis (EDA)
Data visualisation and storytelling
